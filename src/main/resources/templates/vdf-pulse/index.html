Inmetro's Randomness Beacon

Inmetro's Randomness Beacon is a web service, part of the SICCCiber project's effort to generate reliable and public sources of random numbers

About this page

Combinação de beacons utilizando funções de computação lenta.

Este serviço funciona da seguite forma:

  • A cada 5 minutos uma nova submissão é aberta(Ex.: 13:30 - 13:35). O período de submissão dura 60 segundos.
  • Após o término do periodo de submissão, o sistema adiciona o precommit do próximo número juntamente com as contribuições anteriores.
  • O sistema inicia Sloth com "X" iterações.
  • Os resultados são publicados juntamente com os parÃmetros de configuração

The service is available via a public REST API.


Previous Pulse:
/beacon/2.0/pulse/vdf/previous/<timestamp>
Example:

Next Pulse:
/beacon/2.0/pulse/vdf/next/<timestamp>

Last Available Pulse:
/beacon/2.0/pulse/vdf/last
Example:

First Pulse:
/beacon/2.0/pulse/vdf/first
Example:

Verify Pulse:
/beacon/2.0/pulse/vdf/verify?y=<Y>&x=<X>&iterations=<iterations>
Example:

If a request for a Pulse results in no Pulse found, a 404 response is returned.
If a request for a Pulse is not in RFC 3339 format, a 400 (BAD REQUEST) is returned.
